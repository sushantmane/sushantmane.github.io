Difference between URI, URL, URN and Domain-name

Mar 16, 2015

Domain Name

Domain name represents an Internet Protocol(IP) resource, such as a personal computer used to access the internet, a server computer hosting a website, or the website itself or any other services communicated via internet.
Domain names are placeholders for an IP addresses.
For example:
http://sushantmane.github.io/ represents IP address say 103.245.222.133.

URI

URI(Uniform Resource Identifier) identifies a resource either by location(URL), or a name(URN), or both(URL and URN).
URI = URL + URN
For example:
http://facebook.com/sushantmane7
ftp://10.10.7.14/

URL

URL(Uniform Resource Locator) locates as well as identifies a resource.
URL defines the network location of a specific resource and states how that resource can be obtained.
For example:
http://sushantmane.github.io/ is a URL with location sushantmane.github.io of resource and http is a protocol to access that resource.

ftp://10.10.7.14/ug2011/n2/sushant.pdf is a URL with location 10.10.7.14/ug2011/n2/sushant.pdf of resource and ftp is a protocol to access that resource.

Note: All URLs are URIs. But all URIs are not URLs.

URN

URI(Uniform Resource Name), a type of URI, basically states what something is, but do not have information on how to access it.
For example:
author: James F. Kurose
title: Computer Networking: A Top-Down Approach
ISBN-10: 8131790541 this is URN and is pointed by URL to Computer Networking: A Top-Down Approach
